# AI分身市场 - 启动说明

## 项目概述
这是一个AI分身市场平台，允许用户：
- **创作者**：训练AI分身并出租，赚取被动收入
- **使用者**：雇佣AI分身完成各种任务
- **核心机制**：AI做80%工作，真人审核20%，责任关联

## 技术栈
- Next.js 14 + React 18 + TypeScript
- Tailwind CSS
- 本地JSON存储（B方案）

## 已完成的页面

### 1. 首页 (`/`)
- 平台介绍
- 核心特性展示
- 统计数据
- 引导用户创建分身或浏览市场

### 2. 分身市场 (`/client/market`)
- 分身列表展示
- 搜索和筛选功能
- 分身详情卡片（评分、价格、技能等）
- 雇佣入口

### 3. 创作者中心 (`/creator/dashboard`)
- 收益统计（今日/本周/本月）
- 待审核任务列表
- 表现最佳的分身
- 最近完成的任务

### 4. 创建分身 (`/creator/avatar/create`)
- 四步创建流程：
  1. 基本信息（名称、简介、头像）
  2. 人格配置（沟通风格、主动性、专业领域）
  3. 记忆文件（SOUL.md、MEMORY.md上传）
  4. 定价设置（按任务/订阅、能做/不能做）

### 5. 工作空间 (`/client/workspace`)
- 实时对话界面
- 人机协同工作流展示
- 消息发送和接收

## 启动步骤

### 方式1：使用Node.js直接运行

```bash
# 进入项目目录
cd D:/ai-avatar-market

# 安装依赖
npm install

# 开发模式启动
npm run dev

# 构建生产版本
npm run build
```

### 方式2：使用npx（无需全局安装）

```bash
cd D:/ai-avatar-market

# 安装依赖
npm install

# 启动开发服务器
npx next dev
```

## 核心概念

### 人机协同（80/20）
1. **AI阶段**：分身自动与客户沟通，完成任务（80%）
2. **真人审核**：创作者收到通知，审核AI产出（20%）
3. **最终交付**：确认后交付给客户

### B方案架构（当前）
- 创作者自托管AI分身
- 本地JSON文件存储
- 平台只负责撮合和支付

### A方案预留（未来）
- 平台统一托管AI分身
- 可无缝迁移到PostgreSQL
- 支持更复杂的查询和分析

## 定价模型

- **按任务**：每个任务单独定价（¥5-¥10000）
- **订阅制**：按月/年订阅（¥29-¥2999/月）

## 平台抽成

- 平台抽成：10-30%
- 创作者收益：70-90%
