# AI分身市场 - 全端口开发设计文档

## 项目目标
并行开发5大端口：供给端、需求端、管理端、数据端、认证端，构建完整的AI分身市场平台。

---

## 五大端口详细设计

### 1. 供给端 (Supply Side)

#### 用户旅程
```
注册 → 登录 → 个人中心 → 入驻申请 → 审核 → 训练分身 → 上架 → 接受雇佣 → 收益提现
```

#### 功能模块

**1.1 认证中心**
- 注册/登录/找回密码
- 手机号/邮箱验证
- 第三方登录（微信/钉钉）

**1.2 个人中心 Dashboard**
- 欢迎卡片（今日数据概览）
- 快捷操作（创建分身、查看任务、提现）
- 通知中心

**1.3 入驻申请系统**
- 入驻引导页
- 资料填写（基本信息、工作经历、技能标签）
- 材料上传（简历、作品集、AI训练语料）
- 审核状态追踪（待审核/审核中/已通过/已拒绝）
- 审核反馈查看

**1.4 AI分身管理**
- 分身列表（状态：草稿/审核中/已上架/已下架）
- 创建分身向导
  - Step1: 选择训练方式（平台训练/自上传）
  - Step2: 填写分身信息（名称、简介、定价）
  - Step3: 上传训练材料/语料
  - Step4: 预览与提交审核
- 分身编辑/删除
- 分身数据分析（浏览量、雇佣次数、收入）

**1.5 任务中心**
- 待处理任务
- 进行中任务
- 已完成任务
- 任务详情（项目描述、雇佣方信息、交付物）

**1.6 收益中心**
- 收入概览（今日/本周/本月/累计）
- 收入明细
- 提现申请
- 提现记录
- 分成比例说明

**1.7 消息中心**
- 系统通知
- 任务消息
- 雇佣方私信
- 审核反馈

---

### 2. 需求端 (Demand Side)

#### 用户旅程
```
浏览市场 → 筛选分身 → 查看详情 → 发起雇佣 → 付费 → 项目协作 → 成果交付 → 评价
```

#### 功能模块

**2.1 市场首页**
- 推荐分身（基于热度/评分）
- 分类浏览（行业/技能/价格区间）
- 搜索功能
- Banner轮播

**2.2 分身详情页**
- 分身头像/名称/简介
- 评分/雇佣次数
- 技能标签
- 定价（按时/按次）
- 详细介绍
- 用户评价
- 相似推荐
- 「立即雇佣」按钮

**2.3 雇佣流程**
- 雇佣确认页
  - 分身信息确认
  - 服务类型选择（咨询/项目）
  - 付费方式选择（按时/按次/包项目）
  - 项目需求描述
  - 预估费用
- 支付页面（模拟）

**2.4 项目管理中心**
- 项目列表（进行中/已完成）
- 项目详情页
  - AI对话界面
  - 项目进度追踪
  - 文件/交付物管理
  - 联系分身所有者
- 项目完成确认

**2.5 个人中心**
- 我的订单
- 我的项目
- 消费记录
- 账户余额/充值

---

### 3. 管理端 (Admin)

#### 功能模块

**3.1 入驻审核管理**
- 待审核列表
- 审核详情（申请人资料、材料查看）
- 审核操作（通过/拒绝+原因）
- 审核历史

**3.2 分身管理**
- 分身列表（全部/待审核/已上架/已下架）
- 分身详情
- 上下架操作
- 权重/优先级调整

**3.3 用户管理**
- 用户列表（供给端/需求端/管理员）
- 用户信息查看
- 账号状态管理（正常/封禁）
- 封禁记录

**3.4 接口授权管理**
- API密钥管理
- 权限分配
- 调用日志

**3.5 内容审核**
- 分身内容审核
- 用户评价审核
- 违规内容处理

---

### 4. 数据端 (Data Dashboard)

#### 功能模块

**4.1 实时数据看板**
- 今日关键指标（PV/UV/注册/入驻/雇佣）
- 实时在线人数
- 实时交易动态

**4.2 用户数据分析**
- 注册趋势图
- 用户留存分析
- 用户画像分布
- 供给端/需求端比例

**4.3 分身数据分析**
- 分身增长趋势
- 热门分身排行
- 分身活跃度
- 分身收入排行

**4.4 交易数据分析**
- 交易额趋势
- 成交量统计
- 平台分成统计
- 退款/纠纷统计

**4.5 财务数据**
- 收入报表
- 支出报表（分成/提现）
- 利润分析
- 资金流水

---

### 5. 认证端 (Certification)

#### 功能模块

**5.1 身份认证服务**
- 认证申请入口
- 资料上传（身份证/工牌/学历证明）
- 认证状态查询
- 认证标识展示

**5.2 知识产权认证**
- AI分身版权登记
- 训练语料所有权声明
- 数字证书生成
- 认证证书展示

**5.3 1V1咨询服务**
- 专家列表
- 预约咨询
- 在线咨询（文字/语音/视频）
- 咨询记录
- 付费结算

**5.4 增值服务商店**
- 服务列表
- 购买服务
- 我的服务

## 技术架构

### 前端
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI库**: shadcn/ui
- **状态管理**: React Context + Zustand
- **图表**: Recharts

### 部署
- **平台**: Vercel
- **方式**: 静态导出 (SSG)
- **数据**: Mock数据（后续接入API）

## 设计原则

1. **移动端优先** - 所有页面响应式设计
2. **一致性** - 统一的设计语言、交互模式
3. **渐进增强** - 基础功能可用，高级功能逐步完善
4. **数据驱动** - 关键页面都有数据展示
